## Command lines
### Install Node.js:
```
brew install node@20 (version 20)
```
```
brew link node@20
```

### Install Express:
```
npm install express-generator -g
```
```
npx express project-name --no-view
```

### Launch the server after updates:
```
npm run start
```

### Automaticaly launch the server after every update:
```
node --watch
```
